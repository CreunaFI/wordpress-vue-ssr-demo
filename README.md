# Vue and WordPress Server Side rendering demo

## Requirements

* Wordpress 5.0 or newer
* Advanced Custom Fields Pro 5.8 or newer
* Node.js
* Composer

## Install

1. Clone into plugins directory
2. Run `npm install`
3. Run `composer install`
4. Run `npx webpack -w`
5. Activate plugin
6. Add the block to a page or post
